Telegram-бот для управления мероприятиями, регистрации участников, продажи билетов и аналитики.

---

## Возможности

| Функция | Описание |
|---------|----------|
| 📋 Список событий | Пагинированный список предстоящих мероприятий |
| 📌 Карточка события | Название, описание, дата/время, место, свободные места, рейтинг |
| ✅ Регистрация | Регистрация в один клик с подтверждением |
| 🎫 Типы билетов | Обычный, VIP, Бесплатный — каждый со своей ценой и квотой |
| ⏳ Лист ожидания | Автоматическое продвижение при отмене другим участником |
| 🎟 Мои регистрации | Просмотр и отмена своих регистраций |
| 📂 Категории | Концерт, мастер-класс, митап, спорт, обучение и т.д. |
| 🔍 Поиск | По тексту, категории, датам, популярности |
| 🔄 Повторяющиеся события | Еженедельные, двухнедельные, ежемесячные — авто-создание |
| ✅ Чекин | Уникальный 6-символьный код, `/checkin ABC123` |
| ⭐ Отзывы | 1-5 звёзд + комментарий после завершения события |
| 📷 Фотогалерея | До 5 фото на событие, отображение media-группой |
| 🔔 Напоминания | Автоматические за 24ч и 2ч до события |
| 👤 Профиль | Статистика посещений, любимая категория, история |
| 📊 Аналитика | Общая и per-event: посещаемость, выручка, рейтинг |
| 📥 CSV-экспорт | Выгрузка списка участников с типом билета и статусом |
| ⚙️ Админ-панель | Создание, редактирование и отмена мероприятий |
| 👥 Управление участниками | Просмотр списка участников и листа ожидания |
| 📢 Рассылка | Отправка сообщений участникам конкретного события |
| 🔔 Уведомления | Авто-уведомления при отмене, продвижении из очереди |
| 🕐 Автостатус | Прошедшие события автоматически помечаются как завершённые |

## Быстрый старт

```bash
# 1. Клонировать репозиторий
git clone https://github.com/your-username/eventhub-bot.git
cd eventhub-bot

# 2. Установить зависимости
pip install -r requirements.txt

# 3. Настроить переменные окружения
cp .env.example .env
# Отредактируйте .env: добавьте BOT_TOKEN и ADMIN_ID

# 4. Запустить бота
python bot.py
```

## Команды пользователя

| Команда | Описание |
|---------|----------|
| `/start` | Главное меню |
| `/help` | Справка |
| `/profile` | Профиль и статистика |
| `/search` | Поиск событий |
| `/review <ID>` | Оставить отзыв о событии |

## Админ-команды

| Команда / Кнопка | Описание |
|-------------------|----------|
| ⚙️ Админ-панель | Открыть панель администратора |
| ➕ Создать событие | Пошаговое создание (название, описание, дата, время, место, макс. участников, категория, повторение, фото) |
| ✏️ Редактировать | Изменить любое поле, включая категорию и повторение |
| 🚫 Отменить событие | Отмена с уведомлением всех участников |
| 👥 Участники | Просмотр списка участников и листа ожидания |
| 📢 Рассылка | Отправка сообщения участникам события |
| 📂 Категории | Создание и удаление категорий |
| 🎫 Типы билетов | Добавление типов билетов к событию |
| 📷 Фото | Добавление фотографий к событию (до 5 шт.) |
| ✅ Чекин | Просмотр статистики + `/checkin КОД` |
| 📥 Экспорт CSV | Выгрузка списка участников в CSV |
| 📊 Статистика | Per-event: регистрации, посещаемость, рейтинг, выручка |
| `/analytics` | Общая аналитика: все события, категории, выручка |
| `/checkin КОД` | Отметить участника по коду |

> Для получения прав администратора добавьте свой Telegram ID в переменную `ADMIN_ID` в файле `.env`. Можно указать несколько ID через запятую.

## Планировщик задач

Бот использует APScheduler для автоматических задач:

| Задача | Интервал | Описание |
|--------|----------|----------|
| Завершение событий | 10 мин | Прошедшие события помечаются как `completed` |
| Напоминание 24ч | 15 мин | Уведомление зарегистрированных за сутки до события |
| Напоминание 2ч | 15 мин | Уведомление зарегистрированных за 2 часа до события |
| Повторяющиеся события | 30 мин | Создание следующего события для recurring |
| Запрос отзывов | 1 час | Промпт для оценки завершённых событий |
